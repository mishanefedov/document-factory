# workspace/conventions

Your private rules that the agent consults on every doc build.

- `writing-voice.md` — banned/preferred language, tone register, audience-specific framings. See `instructions/writing-voice-guide.md` for how to author.
- `data-disclosure.md` — sample-size handling, forward-looking-claim rules, consent requirements. See `instructions/data-disclosure-guide.md` for how to author.
- `audience.md` — archetypes of the people you write for, with their attention spans, prior knowledge, and decision criteria.

These evolve. Update them after every feedback cycle or council decision that reshapes how you write.
