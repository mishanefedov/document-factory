# workspace/assets

Your brand assets: logos, fonts (self-hosted if needed), images.

Typical contents:
- `logo.svg` — primary logo
- `symbol.svg` — icon-only variant
- `favicon.svg` — for in-document small marks
- `fonts/` — self-hosted font files if you don't rely on Google Fonts CDN

Reference from components via relative paths (`../../assets/logo.svg`).
