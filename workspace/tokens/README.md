# workspace/tokens

Your resolved CSS tokens. Linked by every template in `workspace/templates/`.

- `brand.css` — colors, fonts, typography scale, base element styles. Brand-specific.
- `print.css` — `@page` size/margin, `break-inside` rules. Mostly brand-agnostic.

Scaffolded at setup from `starter-templates/tokens/*` with `{{PLACEHOLDERS}}` resolved to your brand values. See `instructions/brand-token-protocol.md` for naming conventions.
